# Workshop
## Scenario
A client from the Premier League has called, they want a new fixture generator.
### Key Points
- There are 20 teams
- Each teams plays each other home and away
- Teams can only play each other once in each half of the season
- Key dates need to be reserved (i.e. Xmas day)
- The season start on the second saturday of august and ends the 3rd Sunday in May
